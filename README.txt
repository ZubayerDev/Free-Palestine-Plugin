This plugin allows WordPress site admins to display Free Palestine image stickers or banners to raise awareness and show support. Easy to use and customizable.

== Description ==

Show your solidarity with Palestine by adding customizable stickers or banners on your WordPress site. This lightweight plugin enables website admins to select from a list of Free Palestine images and place them on any part of their website (e.g., corners, headers, or footers). Perfect for blogs, news portals, and personal websites supporting humanity.

**Features:**
- Multiple Free Palestine sticker designs
- Mobile responsive
- No coding required – just activate and select your design

== Installation ==

1. Upload the plugin folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to **Settings > Free Palestine** to choose and configure the sticker.
4. Save and see it live on your site.

== Frequently Asked Questions ==

= Can I use my own custom sticker? =
Currently, the plugin offers pre-designed stickers. Custom upload support is coming soon.

= Will it affect my website speed? =
No, the plugin is very lightweight and optimized.
